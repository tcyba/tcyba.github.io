下面讲的是CPU与外部器件相连！

看了看汇编！

汇编以机器指令是一一对应的！

是机器指令的助记符！

0000 0000 0000 0000 二进制

0000 16进制

0000 0000 0100 0000 0001 0000 0101 1101 8*4 = 32位

0 0 4 0 1 0 5 D

那64位的岂不更长！转成16进制也长！

基础知识

1汇编指令

2 伪指令

3 其他符号

存储器

存储器，计算机里每个部件都有存储器！

网卡里有存储器，显卡里有有存储器！等等

内存是一个主要的存储器！

CPU – 内存 – 磁盘

游戏从磁盘加载到内存！CPU才能处理！

虚拟内存是从磁盘里调用的！

指令数据是自己控制的！

存储单元

CPU对存储器的读写

就是CPU通过导线！连接(网卡)这给芯片！

地址总线 -》 存储器(内存)地址！

数据总线 –》 数据

控制总线–》执行的操作！

读存储单元的地址！命令|数据|

通过走不同的线路来区分是什么 指令还是数据还是地址！

还有是下高上低！

地址总线 – 》就是让内存定位到那个地址！！!

数据总线 –》 就是传输数据！ 觉得传输的速度！

控制总线 –》 传输的每个位置！执行什么操作！

小结

主板

接口卡

如CPU控制网卡！ 外部你在插入网线！

还有usb接口！CPU控制usb接口！外部你插入U盘！

就是CPU可以控制那个接口！

存储器

随机存储器是–》内存！关机就没了！

ROM里的BIOS

BIOS就是电脑启动最开始，主板运行的程序！如：检查你插鼠插没！启动CPU什么的！

像网卡|显卡 也有BIOS!

各种卡都有BIOS！

大致流程

内存

每个卡里面都有存储器！(这个应该把！🙄学的不精)

网卡里的存储器通过总线相连CPU！

看完这句话好像懂了！

把所有存储器里的存储单元 都 统一到 逻辑存储器里！(这个逻辑存储器是不存在的！只是方便理解！)

这样我们平常所说的内存！就是这个逻辑存储器！他就是把所有卡的存储器 汇总在一起了！(其实不存在！)

他受到我们CPU寻址能力的限制！
